 카드뉴스 메이커 — 사용법 & 팀 공유 (비개발자용)
 파일: card-news-maker.html  (이 파일 하나면 끝)
==========================================================

[이게 뭔가요]
 - HTML 파일 하나로 동작하는 카드뉴스 생성기입니다.
 - 설치·터미널·AI 필요 없음. 더블클릭하면 브라우저에서 열립니다.
 - 팀원에게는 이 파일(card-news-maker.html) 하나만 보내주면
   똑같이 사용할 수 있습니다. (카톡/메일/드라이브 첨부 OK)

[쓰는 법]
 1) card-news-maker.html 더블클릭 → 브라우저(크롬/엣지)에서 열림
 2) 왼쪽 폼을 채웁니다:
      - 핸들: 본인 계정 (@나의계정) 으로 바꾸세요
      - 날짜 / 핵심 한 줄 / 툴 체인
      - 이미지: 점선 박스에 끌어다 놓거나 클릭해서 선택 (여러 장 OK)
                썸네일에서 순서(↑↓)·삭제(×) 가능. 1장 = 카드 1장.
      - 워크플로우 단계: "+ 단계 추가" 로 늘리기
      - 인사이트 / 해시태그 (#SSUPERWASABI 는 자동으로 붙음)
 3) 오른쪽에 카드가 실시간으로 만들어집니다.
 4) 저장:
      - 카드에 마우스 올리면 뜨는 "↓ PNG" = 그 카드 1장 저장
      - 좌상단 "↓ 전체 다운로드 (ZIP)" = 전체를 ZIP 하나로 저장
 5) 저장된 PNG들을 인스타/스레드 등에 업로드.

[카드 구성]
 표지 → 이미지(1장당 1카드) → 워크플로우 단계 → 인사이트 → 마무리
 - 이미지는 자동으로 1:1에 맞게 가운데 크롭됩니다 (비율 안 깨짐).
 - 전부 1080x1080 정사각으로 저장됩니다.

[알아두면 좋은 점]
 - 글꼴(Inter)은 인터넷에서 불러옵니다. 인터넷이 없으면 기본 글꼴로
   보이지만 동작은 됩니다. (저장 기능은 인터넷 없이도 작동)
 - 크롬/엣지 권장. (사파리도 대체로 동작)
 - 전체 다운로드는 ZIP 1개라서, 브라우저의 "여러 파일 다운로드" 차단에
   걸리지 않습니다.

[팀 공유 방법 — 셋 중 아무거나]
 A) 파일 그대로 전송: card-news-maker.html 을 메일/메신저로 보냄
 B) 공용 드라이브(구글드라이브/원드라이브)에 올려두고 각자 다운로드
 C) (나중에) 웹주소로 열고 싶으면 Leo 에게 "온라인으로 올려줘" 요청
    -> GitHub Pages 같은 무료 호스팅으로 URL 만들어 줄 수 있음
