# One Prompt a Day — 카드뉴스 생성기 개선 (요약)

**작성:** @ssuperwasabi · **갱신:** 2026-06-02 · **상태:** 완료·검증 통과

## 1. 한 줄 요약
AI 작업물(이미지)과 워크플로우를 **인스타/스레드용 카드뉴스(캐러셀)** 로 자동 변환하는 생성기. 이번 개선으로 **입력 이미지 비율에 카드가 자동으로 맞춰지고, 이미지가 잘리지 않고(무크롭) 들어가도록** 전면 개편했습니다.

## 2. 배경 / 문제
- 기존: 카드가 **1:1 고정**이라 입력 이미지를 **강제로 잘라(crop)** 채움 → 공들인 작업물의 가장자리 손실, 비율 불일치.
- 목표: ① 입력 이미지 비율에 **카드 크기 자동 조정** ② 이미지 **무크롭** 으로 온전히 표시.

## 3. 개선 내용
| 항목 | Before | After |
|------|--------|-------|
| 카드 비율 | 1:1 고정 | **입력 이미지 비율에 자동 맞춤** (1:1, 4:5 등) |
| 이미지 처리 | 강제 크롭 | **Fit(무크롭)** — 전체 보존, 남는 공간은 여백 |
| 비율 왜곡 | 발생 가능 | **0** (해결·실측 검증) |
| 적용 범위 | 생성기 | 생성기 + **브라우저 메이커** 동일 적용 |

- **자동 비율 모델:** `카드 = 이미지 영역 + 캡션 바`. 입력 비율로 카드 높이를 계산하고, 인스타 안전 범위(4:5~1.91:1)로 제한.
- **결과:** 1:1 → 1080×1200, 3:4 → 1080×1350, 텍스트 전용 → 1080×1080 (모두 자동).

## 4. 검증 결과 (헤드리스 자동 테스트, 전부 통과)
- 1:1 이미지가 영역을 **가로·세로 100% 채움**, 비율 **무왜곡**, export 정확.
- 실제 18장 덱: **전 카드 규격 일치 + 이미지 무왜곡 + 전체 ZIP 다운로드 정상.**
- 브라우저 메이커: 이미지 드롭 시 **비율 자동 변경 + 무크롭** 확인. (실사용 검수 완료)

## 5. 사용 방법
1. **브라우저 메이커(`card-news-maker.html`)** 열기 — 설치 불필요, 파일 하나로 동작.
2. 텍스트(핵심 한 줄·툴 체인·워크플로우·해시태그) 입력 + 이미지 **드래그&드롭**.
3. 오른쪽에 카드가 **입력 비율대로** 자동 생성 → 카드별 **PNG** 또는 전체 **ZIP** 저장.

## 6. 결과물 / 산출 가치
- AI 콘텐츠 1건을 **즉시 SNS 캐러셀로** 전환 (디자인 통일·무크롭·무왜곡).
- 일관된 **StudioBlank** 톤(모노크롬·미니멀)으로 브랜드 일관성 확보.
