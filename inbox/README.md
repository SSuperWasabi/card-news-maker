# inbox — 카드뉴스 입력 폴더

새 1pd 콘텐츠는 **포스트당 폴더 하나**로 여기에 모읍니다.
폴더를 채운 뒤 Leo에게 **"이 폴더로 카드뉴스 만들어줘: …\inbox\<폴더명>"** 라고 하면 됩니다.

## 폴더 만드는 법

1. `_TEMPLATE` 폴더를 복사해서 `YYYY-MM-DD-슬러그` 형태로 이름 변경
   (예: `2026-05-30-harbor`)
2. 안의 `post.md`를 채우기
3. 생성 이미지/영상 스틸을 같은 폴더에 넣기

## 폴더 구성

```
inbox/2026-05-30-harbor/
  post.md          ← 글/툴체인/워크플로우 메모 (필수)
  cover.png        ← 메인 히어로 생성 이미지
  01.png 02.png …  ← 추가 생성 이미지
  video-still.png  ← 영상에서 뽑은 대표 프레임
  notes.txt        ← 부가 메모 (선택)
```

## 소스별 주의

- **텍스트**: `post.md`에 적거나, 대화창에 붙여넣어도 됨
- **이미지**: 반드시 **파일로** (카드에 base64로 박아 넣음). 가장 강한 1~2장만 골라도 OK
- **영상**: Leo는 영상을 못 봄 → **대표 스틸 1장 + 텍스트 설명**으로. 카드뉴스도 정지 이미지라 프레임이 답
