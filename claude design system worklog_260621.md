# 작업 로그 — 카메라 UI 카드뉴스 템플릿

## 프로젝트 개요
- 파일: `카드뉴스 템플릿.dc.html`
- 디자인 시스템: COM2US Store Design System (시각 스타일 바인딩)
- 대상: One Prompt a Day 카드뉴스 (1080 × 1350, 5장 흐름)

---

## 작업 이력

### 1. 마지막 CTA 카드(05/05) 디벨롭
첨부 초안(링크 디렉터리형 CTA)을 받아 디자인 디벨롭. 요청 핵심:
**레이아웃 안정감 + 세련된 텍스트 타이포**.

#### 변경 내역
**타이틀**
- Inter 800, letter-spacing `-0.045em`, line-height `0.96`
- "One Prompt / a Day." 2행 분할 → 무게중심을 위로
- 마침표만 40% 톤 악센트 (모노크롬 유지, 새 색 추가 없음)

**링크 디렉터리**
- 좌 플랫폼명(Inter 600) / 우 핸들(IBM Plex Mono) 2열 구조
- 행마다 1px 룰 + 동일 28px 패딩 → 수평 기준선이 잡힌 테이블
- 베이스라인 정렬(`align-items:baseline`)
- 항목: Instagram / Facebook / Threads / YouTube

**리듬 · 안정감**
- 헤더 · (타이틀+디렉터리) · 푸터 3존을 `space-between`으로 고정
- 상단 라벨과 하단 워드마크가 대칭 앵커 역할
- 본문 분량이 바뀌어도 프레임이 흔들리지 않는 구조

**시스템**
- 차콜 `#0A0A0A` · radius 0 · 무그림자 · 1px 라인
- 흰색 위 투명도(46~52%)로 위계만 분리

#### 반영 위치
1. 보드 상단에 큰 미리보기(540×675) + 디벨롭 노트 섹션 추가
2. 5장 흐름의 05번 썸네일도 동일하게 갱신
